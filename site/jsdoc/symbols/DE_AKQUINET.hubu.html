<static> DE_AKQUINET.hubu.bind(binding)
Binds a component to another component.
Parameters:
{Object} binding
this object describes the binding. It contains:
  • component : the component to inject ({String} or {DE_AKQUINET.AbstractComponent} object)
  • to : the component receiving the 'component' ({String} or {Component}DE_AKQUINET.AbstractComponent} object)
  • into : where the component is injected, it can be either a function (called on 'to' with 'component' as parameter), or a {String}. In this latter case,'to'.'into' is injected. If it's a function the function is called, if it's an object it is just injected (assigned).
  • contract
    • : The binding contract (i.e. interface) (optional). The contract is a simple object defining the methods that the destination can invoke on the source. If the source is not conform to the contract, the binding is rejected. The contract is a kind of interface. By default a proxy implementing the contract is injected enforcing the contract-based interaction. However, you can disable the proxy by setting proxy:false
  • proxy
    • : boolean indicating if proxies are enabled or disabled. By default they are enabled if contract is set, however, setting proxy to false enables the injection of a direct reference.
Returns:
the current hub

<static> DE_AKQUINET.hubu.registerComponent(component, configuration)
Registers a new component on the hub. If the component already exists, the registration is ignored. The lookup is based on the 'getComponentName' method. This method allows to configure the component.Once successfully registered, the hub call the 'configure' method on the component passing a reference on the hub and the configuration to the component. If component is null, the method throws an exception.
Parameters:
{DE_AKQUINET.AbstractComponent} component
the component to register
{Object} configuration
the component configuration (optional). If the configuration contain the 'component_name' key, the component takes this name.
Returns:
the current hub

<static> DE_AKQUINET.hubu.unregisterListener(component, callback)
Unregisters listeners for the given component. According to the arguments, several cases can happen: 'component' can be either a String of a Component. In case of the string, we look up for the component using 'getComponentName'. If 'callback' is defined, only the listener matching component and callback will be unregistered. Otherwise all listeners of the component will be unregistered. If 'component' is null, this methods does nothing.
