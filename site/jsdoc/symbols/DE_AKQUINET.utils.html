<static> DE_AKQUINET.utils.isObjectConformToContract(object, contract)
Checks that the given object is conform to the given contract The contract is a javascript object. The conformity is computed as follow: O is conform to C if and only if for all i in C where C[i] != null O[i] != null && typeof(C[i]) = typeOf(O[i]) This is an implementation of 'Chi': Metamodel <- chi <- Model -> mu -> System where chi is : isObjectConformToContract and mu is representationOf.
Parameters:
object
the object to check
contract
the contract
Returns:
true if the object is conform with the given contract, false otherwise.
